# Security Policy

This repository contains experimental proof-of-concept code developed and
published by Parity. It is provided for research, experimentation, and
developer education only. It has not been audited and may contain bugs,
vulnerabilities, or incomplete features. Use at your own risk.

## Reporting a vulnerability

If you discover a bug, exploit, or similar vulnerability, you are welcome to
report it to us through either of these public channels:

- **GitHub private vulnerability reporting** — use the **Report a vulnerability**
  button under the repository's **Security** tab.
- **Public issue tracker** — open an issue in this repository for non-sensitive
  reports.

Please include enough detail to reproduce the issue.

## What to expect

Because this is experimental proof-of-concept code, we make no commitment to
provide fixes, patches, or any particular response time, and we do not operate
or guarantee any deployment based on this code. Reports are reviewed on a
best-effort basis.
